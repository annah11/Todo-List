# Todo List App

This is a simple Todo List application built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Delete tasks
- List tasks
- edit tasks
- save tasks

## How to Run

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser.
